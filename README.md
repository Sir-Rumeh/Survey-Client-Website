# Survey Plus Client Website

Next.js + TypeScript landing page scaffold implementing the provided design.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run development server

```bash
npm run dev
```

Notes:

-    Tailwind and CSS variables are configured in `src/styles/globals.css`.
-    Toggle dark mode by clicking the theme button in the navbar (persisted to `localStorage`).
